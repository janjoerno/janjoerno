History of JAVASCRIPT

JavaScript was created by Brendan Eich in 1995 during his time at Netscape Communications. It was inspired by Java, Scheme and Self. Netscape, for a time, made the best browser in the world and enjoyed market dominance.

Brendan Eich developed JavaScript at Netscape Communications in 1995. They created JavaScript as a scripting language for use with Netscape Navigator, the company's flagship web browser. Because Netscape owns the name "JavaScript," when it was added to Internet Explorer, it had to be officially called "JScript."
